For making flux plots, there are three important files here. First, 'flux_plots.py' has a few useful functions.

The functions 'plot_alpha', 'colorcolor_plot', and 'plot_SEDs' make plots given the desired source IDs and bands (when applicable). They use the data from 'IR_matched_catalog_B7.fits', which is based on the original images. 

The function 'multi_img_SED' is the main, updated function. It takes the name of the desired image in each band, and creates an SED with Forbrich upper limits, Forbrich alpha measurements, and cutouts in each band. The script 'make_SEDs.py' creates these SEDs for all sources with detections in all bands (B3, B6, B7). 

Lastly, 'luminosity_calc.py' calculates the equivalent luminosity and brightness temperature for each source detected in all 3 bands. 

Also, 'deconv_sources.py' gives the indices of sources which can be deconvolved.
